### 局部变量和成员变量
| 不同       | 局部变量                                     | 成员变量                                     |
| ---------- | -------------------------------------------- | -------------------------------------------- |
| 定义的位置 | 在方法的内部                                 | 在方法的外部，直接写在类中                   |
| 作用范围   | 只有方法中才可以使用，出了方法就不能使用了   | 整个类全都可以通用                           |
| 默认值     | 没有默认值，如果要使用，必须收手动创建初始值 | 有默认值，规则和数组一样                     |
| 内存的位置 | 位于栈内存                                   | 位于堆内存                                   |
| 生命周期   | 随着方法进栈而诞生，随着方法出栈而消失       | 随着对象创建而诞生，随着对象被垃圾回收而消失 |
